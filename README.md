## About Tani Merdeka

Tani merdeka merupakan sistem informasi pengelolaan data anggota tani merdeka, platform berbasis website dan android. Membantu dalam pengoptimalisasi pengelolaan data anggota tani satu provinsi jawa tengah

## 🔀 Branch Rules
1. Nama branch (Nama_Fitur) contoh : Nana_search, Nopi_Beranda
2. Push **wajib ke branch masing"**
3. Wajib review sebelum merge ke branch master

## 📜 Panduan Commit Message (Git Convention)
1. Format Dasar
- tipe (scope): (subjek/judul singkat) => 
**CONTOH : feat(auth): tambahkan login menggunakan NIK | fix(wilayah): perbaiki bug dropdown kecamatan tidak muncul**

2. Tipe Commit
- feat(Feature) : Menambahkan fitur baru. **Contoh:** Tambah form input petani, bikin endpoint API baru.
- fix(Bugfix) :  Memperbaiki error/bug. **Contoh:** Fix error null pointer, benerin typo di UI.
- docs(Documentation) :  Hanya merubah dokumentasi. **Contoh:** Update README, nambah komentar kodingan.
- style(Formatting) :  Merapikan kode (spasi, koma) tanpa ubah logika. **Contoh:** Format ulang indentasi, hapus spasi kosong.
- refactor(Refactor) : Mengubah kode untuk optimasi tapi outputnya sama. **Contoh:** Bersihin kodingan berantakan, optimasi query DB.
- test(Testing) : Menambah atau memperbaiki unit test. **Contoh:** Tambah test untuk fitur login.
- chore(Maintenance) : Tugas rutin, update library/dependency. **Contoh:** Update versi Laravel, update pubspec.yaml.
- ci(CI/CD) : Perubahan pada script konfigurasi server/deploy. **Contoh:** Edit file github actions, dockerfile.
